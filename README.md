# ✨ Personal Developer Portfolio

Welcome to my personal developer portfolio! 
This is a highly interactive, modern, and performance-optimized Application built to showcase my projects, skills, and journey as a Computer Science Student.

🔗 **Live Demo:** [sneha-portfolio.vercel.app](https://portfolio-snehaportfolio.vercel.app) 

---

## 🚀 Key Features

- **Dynamic Interactive UI:** A heavily customized design system featuring glassmorphism, vibrant gradients, and smooth micro-animations.
- **Tech Stack Showcase:** An interactive "falling tech stack" mini-game built using Matter.js where visitors can interact with physics-enabled tech icons.
- **Background Effects:** Features a WebGL-powered colorful fluid cursor effect that reacts to mouse movements.
- **Top Projects:** A sleek, animated UI to showcase my best work, including KrishiRakshak AI and Cosmic Stride.
- **Live GitHub Stats:** Real-time integration of my GitHub contribution graph using `react-github-calendar`.

---

## 🛠️ Tech Stack

This portfolio was built purely from scratch without relying on heavy CSS frameworks, prioritizing custom design and performance.

### **Core**
- **React.js** – Frontend framework
- **Vite** – Lightning-fast build tool
- **JavaScript** – Core logic
- **CSS3** – 100% custom styling, animations, and media queries

### **Libraries & Tools**
- **Framer Motion** – For seamless page transitions and scroll animations
- **Matter.js** – 2D physics engine for the interactive skills section
- **WebGL Fluid** – For the dynamic background cursor effects
- **React Icons** – Scalable SVG iconography
- **Vercel** – Global edge deployment and hosting
